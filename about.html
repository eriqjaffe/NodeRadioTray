NodeRadioTray

NodeRadioTray is a simple, cross-platform internet radio player that sits in the system tray, hence the name. It is heavily influenced by a Linux application called RadioTray, originally written by Carlos Ribiero.

The original implimentation of this was WinRadioTray, Windows-only C# app. While functional, it has some quirks and bugs that I've never been able to sort out, and I have never been entirely comfortable working with C# anyways. Also it was Windows-only which is fine as far as it goes, but after discovering Node.js and Electron I decided to rework WinRadioTray as a cross-platform app.

This software predominantly makes use of the following:

  • Node.js (hence the name).
  • Electron
  • BASS Audio Library (specfically, a slightly modified version of Psycokwet's implmentation) for audio playback.
  • icecast-parser for metadata retrieval.
  • Icons from Icons8.

If you have any questions, feel free to email me or report an issue on the project's GitHub page.
